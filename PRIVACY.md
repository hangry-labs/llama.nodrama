# Privacy Policy

`llama.nodrama` is a local dashboard for `llama.cpp` servers.

The application does not collect, sell, or transmit user data to Hangry Labs.
It connects only to the `llama.cpp` server URL configured by the user and may
check GitHub Releases for update information.

Dashboard data, request metadata, log tails, and chat test content are processed
locally by the running `llama.nodrama` instance. Users are responsible for the
`llama.cpp` server and log files they configure.
